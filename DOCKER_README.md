# Pythia Conversations - Docker Setup

## 🚀 Quick Start

```bash
docker-compose up --build -d
```

That's it! Frontend: http://localhost:3000

## 🛠️ Commands

```bash
docker-compose up --build -d    # Start
docker-compose down             # Stop
docker-compose logs -f          # Logs
```

Simple and fast! 🚀
